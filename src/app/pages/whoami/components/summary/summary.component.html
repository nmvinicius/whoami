Resumo Profissional

Atuo como DevSecOps na Softfocus, com papel-chave na migração para Kubernetes e na implementação de sistemas de autenticação centralizada. Minha equipe contribuiu para a transição do banco de dados para uma solução gerenciada internamente, reforçando a segurança e eficiência operacional. Meu foco inclui o desenvolvimento de APIs para otimizar processos de desenvolvimento e operações, além de trabalhar no comitê de segurança da informação para fortalecer as práticas de segurança.
